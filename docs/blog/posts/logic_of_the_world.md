# 《世界的逻辑》读书笔记

这是最近在读的一本书，世界的逻辑，作者是南方科技大学的马兆远教授。

为什么开始读这本书呢？是我突然看到了一个短视频，一位up主在飞机的头等客舱采访别人，刚好采访到了马兆远教授。教授的一句话让我感觉非常深刻：“在AI大模型的时代，获取知识的成本是非常低的。但是大模型能够直接给你答案，但是答案的正确与否，却是需要你自己去把握的。”这段话让我认识到，我必须掌握一种方法论，这种方法论能够帮助我在任何时间，任何地点，分析我所获取的信息的有效性，真实性，哪怕当时无法分析，但是能够帮助我在有限信息内，得出一个相对可靠的判断，结论。我看到马兆远教授写了这本《世界的逻辑》，我想应该拜读一下，尝试能否从书中获得一些启发。

这份blog将是我在读《世界的逻辑》的过程中，所得到的一些感悟，摘抄的金句以及自己的联想。
