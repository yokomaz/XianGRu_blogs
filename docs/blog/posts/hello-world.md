# 博客上线了！

折腾了好一阵，终于把这个博客搭起来了。

很久以前就有写博客的想法，但总是因为各种原因搁置。选什么平台、用什么框架、域名怎么搞……想得越多，越动不起来。

这次不一样。我决定**先做，再改**。

## 为什么选 MkDocs

用了 MkDocs + Material 主题，几个原因：

1. **纯 Markdown** — 不用学任何新语法，我平时就在用
2. **Material 主题很美** — 开箱即用，支持明暗切换
3. **静态站点** — 生成 HTML 扔 GitHub Pages 上就行，零成本
4. **Python 生态** — 用 `uv` 管理依赖，干净利落

## 接下来

我打算保持一个舒服的写作节奏 — 不用日更，不用周更，有感而发就好。

种一棵树最好的时间是十年前，其次是现在。:seedling:
