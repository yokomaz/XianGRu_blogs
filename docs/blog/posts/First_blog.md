# 博客上线了！

今天是2026年7月9日，很独特，也是我出生的第10000天，所以想开始利用blog来记录自己的生活。

由于之前并不怎么习惯于记录自己的生活，因为我觉得生活好像一直平平淡淡，每天固定的起床，吃饭，睡觉，日复一日的平静，没有什么记录的必要性。而且我也不是一个善于表达自己的人，可能因为我的MBTI是INFJ？再加上双鱼座的buff，啊哈哈哈，习惯于向内寻求力量并且极度的多愁善感。但是这两年发生了很多事情，让我觉得如果不记录自己的话，那么在未来的某一天，我回想我的生命中如果经历了什么事情，我会一无所知，那这种事情对我来说是可怕的。

这是我第一次用文字的方式记录自己的生活，虽然高考我的语文考的还算正常发挥，但是自高考之后，我就在也没有用过这么长的文字来表达自己的观点了。可能在写简历的时候会有一小段话来介绍自己，但是那毕竟还是为了讨口饭吃。

# 我是谁？
首先在生命的第10000天这个时间点里，我想重新认识一下我自己。

我的id是陈相汝，这个id从我出生就一直伴随着我，如果不出意外的话，也应该会伴随着我直到入土。说实话我感觉我对这个id还是非常满意的。“陈”就不多说了，我的姓氏，听家里长辈说网上追溯的话，能追溯到宋朝的某个家族--谁祖上还没有阔绰过呢？“相汝”的话，其实在小时候没文化的时候，觉得这个名字好像一个女生的名字（虽然也有很多人说他们最开始听到这个名字的时候以为是一个女生），现在的话感觉好多了，相汝的意思就是与你在一起。当然不是说和任何人在感情上在一起，而是说和值得珍惜的人相处，当然每个人对于谁是否值得珍稀有各自的看法，但这并不是这一段文字的主题。

我出生于1999年2月22日，这个日期确实比较好记，因为实在是太2了，我甚至怀疑这导致了我有的时候会表现的很2。当然大部分情况下我还是很正常的，也可能不正常，毕竟在讨论的最开始我们没有对什么是正常人做一个严格的定义。那就故且认为我是一个正常的人吧，四肢健全，没有什么奇奇怪怪的疾病。颜值的话，我给我打一个9/10分，少一分怕我骄傲啊哈哈哈。当然严格意义上讲我肯定是普通人的颜值了，但是人总得对自己有点自信好吧，毕竟在漫长的的人生路上，自我是最忠诚的战友。

# 我的兴趣？
兴趣的话，我比较喜欢运动。具体来说，健身啦，跑步啦，羽毛球啦，但是最喜欢的还是一个叫做跑酷（Parkour）的运动。这是一门学习如何控制身体穿越各种障碍物的运动。我大概从初中开始接触吧，也有上过一些课程，但大部分时间都是自己在练习。在跑酷的过程中，我能感受到自己完全掌控了身体的运动。每一根肌肉线条的收缩，每一次神经细胞的反射，让我能够清晰的认识到我不是一个麻木的生物，而是一个可以随心所以控制自己的灵魂，我就是我的身体和灵魂百分之百的主宰者，这种感觉令我非常的着迷，也给了我能够克服任何困难的底气。当然为了我能够有一个支持我跑酷的身体素质，我也经常去健身房里增强我的身体协调性啊，爆发力啊，柔韧啊等等,所以也算健身房的常客了。

emmmm，抛开运动的话，我对读书，硬件开发，人工智能也很感兴趣。有可能是因为我的专业是电子信息+计算机科学与人工智能，所以我对开发一些小玩意有着一定的涉猎，比如钢铁侠里的贾维斯。不过我在这方面的起步比较晚，现在还处于一个朦朦胧胧的摸索状态，我在后续也会记录一下我的开发之路。

# 我怎么维护这个blog？
我打算保持一个舒服的写作节奏 — 不用日更，不用周更，有感而发就好。我会在这里记录我的学习，我的日常，我的读书，把这里当作一个自己和自己对话的窗口。

**我之前忽视你太久了我的朋友，我们未来一定要好好地走下去！**
